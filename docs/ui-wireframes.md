# UI Wireframes

## Home / Recipe Library

- Header: app name and search input
- Filter chips: all, breakfast, lunch, dinner, quick
- Recipe cards: image, title, servings, meal type
- Bottom navigation: home, add, filters

## Recipe Detail

- Header: back button, recipe name, more menu
- Large recipe image
- Title, meal type, servings, tags
- Source links
- Ingredients
- Ordered steps
- Edit button

## Add / Edit Recipe

- Header: cancel and save
- Recipe name
- Servings stepper
- Meal type segmented control
- Image upload or pasted URL
- Source links list
- Ingredients list
- Steps list

## Filters Sheet

- Meal type radio buttons
- Tag checkboxes
- Ingredient search
- Clear filters
- Done action
