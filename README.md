# Meal Prep Recipe PWA

This project folder contains the MVP plan, architecture diagram, and UI mockups for a private mobile-first recipe app for overseas exchange meal prep.

## Recommended Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Hosting: Vercel free/Hobby tier
- Backend: Supabase free tier
- Database: Supabase Postgres
- Auth: Supabase Auth
- Images: Supabase Storage or pasted image URLs
- Device experience: Progressive Web App added to iPhone Home Screen

## MVP Goal

Create a private personal recipe library that works well on iPhone, lets you save recipes with ingredients, steps, servings, images, and source links, and leaves room for future meal-planning and grocery-list features.

## Included Files

- `docs/mvp-plan.md`: detailed MVP plan and implementation outline
- `docs/architecture.mmd`: editable Mermaid architecture diagram source
- `docs/architecture.svg`: architecture diagram image
- `docs/ui-mockups.svg`: main mobile UI mockups as an image
- `docs/ui-wireframes.md`: text reference for the UI screens

## MVP Screens

- Recipe library home screen
- Recipe detail screen
- Add/edit recipe screen
- Filter sheet

## Future Features

- Weekly meal prep planner
- Grocery list from selected recipes
- Serving scaler
- Cost estimate per serving
- Equipment filters such as rice cooker, microwave, stovetop, no oven
- Exchange-friendly tags such as cheap, one-pot, freezer-friendly, dorm-friendly
- AI-generated placeholder images
- Nutrition/macros
